## 이력서

### 자유 양식 이력서

- 상단 : 이름 / 이메일,깃허브,블로그 / 사진X
- summary :  2-3줄 이내
- 경력 : 최신 순
- Personal Project : 직무 연관성, 기술적 난이도 있는것 넣기
- 발표 슬라이드(기술발표 기회 있으면 하기!)

#### summary : 여태까지의 경험, 현재 관심사, 하고 싶은 것
#### 이력

- 어떤일, 어떤기술, 왜
- 어떤 결과 (성과나 수치 중심)
- 위를 한 문장으로

3. 3순위->2순위->1순위
